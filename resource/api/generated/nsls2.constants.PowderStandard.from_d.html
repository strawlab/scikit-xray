nsls2.constants.PowderStandard.from_d¶

classmethod PowderStandard.from_d(name, d, hkl=None)[source]¶

Method to construct a PowderStandard object from known \(d\) values.

Parameters:

name : str

The name of the standard

d : array

The known plane spacings

hkl : list, optional

List of (h, k, l) tuples of the Miller indicies that go with each measured \(2\theta\). If not given then all of the miller indicies are stored as (0, 0, 0).
Returns:

standard : PowderStandard

The standard object
